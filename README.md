# COS760-Project-G19

This is the repository for Group 19's COS760 (NLP) project for 2026 at the University of Pretoria (Department of Computer Science, EBIT)

## Project Details

**Topic:** Detecting Machine Generated Texts

**Objective:** Develop models to distinguish between human- and machine-generated texts in African languages.

**Key Tasks:**

* Train classifiers to detect machine-generated text.
* Evaluate transfer learning and explainability techniques.
* Analyze linguistic patterns in LLM-generated text.
